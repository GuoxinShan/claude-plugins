# 资源管理命令详细参考

Session 自动从 `~/.openturtle/session.json` 加载，通常无需手动传参。

通用模式：
```bash
python CLAUDE_PLUGIN_ROOT/scripts/otcli.py <resource> <action> [选项]
```

带 `--data` 的操作需要准备 JSON 文件，内容为请求体。

---

## Workflow

```bash
# 列表
python CLAUDE_PLUGIN_ROOT/scripts/otcli.py workflow list --base-url URL --token TOK
# 选项: --status draft/published --keyword KEY --page N --page-size N

# 获取最新已发布版本
python CLAUDE_PLUGIN_ROOT/scripts/otcli.py workflow get --group-id <group_id> ...

# 创建（从 JSON 文件）
python CLAUDE_PLUGIN_ROOT/scripts/otcli.py workflow create --data workflow.json ...
python CLAUDE_PLUGIN_ROOT/scripts/otcli.py workflow from-json --data workflow.json ...

# 更新草稿
python CLAUDE_PLUGIN_ROOT/scripts/otcli.py workflow update-draft --group-id <gid> --data update.json ...

# 发布
python CLAUDE_PLUGIN_ROOT/scripts/otcli.py workflow publish --group-id <group_id> ...

# 版本管理
python CLAUDE_PLUGIN_ROOT/scripts/otcli.py workflow versions --group-id <gid> ...
python CLAUDE_PLUGIN_ROOT/scripts/otcli.py workflow version --group-id <gid> --version N ...
python CLAUDE_PLUGIN_ROOT/scripts/otcli.py workflow new-version --group-id <gid> ...

# 导入导出
python CLAUDE_PLUGIN_ROOT/scripts/otcli.py workflow export --group-id <gid> ...
python CLAUDE_PLUGIN_ROOT/scripts/otcli.py workflow import --file workflow.json ...

# 组合视图 & 变更
python CLAUDE_PLUGIN_ROOT/scripts/otcli.py workflow composition --group-id <gid> ...
python CLAUDE_PLUGIN_ROOT/scripts/otcli.py workflow change-proposal --group-id <gid> --data proposal.json ...
python CLAUDE_PLUGIN_ROOT/scripts/otcli.py workflow change-logs --group-id <gid> ...
```

```bash

  --base-url <base-url> \
```
