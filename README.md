# Claude Plugins

Personal Claude Code plugins and skills collection.

## Plugins

| Plugin | Description | Status |
|--------|-------------|--------|
| [codebase-docs](codebase-docs/) | Automated codebase documentation with progressive disclosure | v0.1.0 |

## Usage

```bash
# Load a specific plugin
claude --plugin-dir d:/dev/claude-plugins/codebase-docs

# Or symlink to project
ln -s d:/dev/claude-plugins/codebase-docs .claude-plugin/codebase-docs
```
